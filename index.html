Tags and Elements

Tags Elements
<p> This is a paragraph
<a> This is an anchor, commonly used to hyperlink
<h1> This is the largest heading size normally used at top of page or for headings when trying to focus attention.
<h2> This is the 2nd largest heading size normally used for paragraph headings etc.
<h3>-<h6> Heading 3 to 6 vary in size from larger to smallest.
<br> This will create a break on the page. Equivalent to pressing return on keyboard.

Attributes

Name Description
href Commonly used to link hyperlinks within a web page or to a stylesheet followed after <a
src Short hand for source, this can be used to source location of where an image or video may be saved and can be used along with elements such as img or video
height Another that can be used within img element to adjust the height of sourced media.
width Another that can be used within img element to adjust the width of sourced media.
alt This attribute provides text if image cannot be displayed.
The value of the alt attribute can be read by screen readers. This way, someone "listening" to the webpage, e.g. a vision impaired person, can "hear" the element.
lang Declared at the HTML element and determines the language. I.E "lang=en-UK"

Tables

Name Description
<table> Table is used to start of the container that will hold your table and requires a closing tag.
<thead> Thead, means "Table Head" and this element defines a set of rows defining the head of the coloumns of the table.
<th> TH defines the header cell within a HTML table. <th> elements are bold and centered by default.
